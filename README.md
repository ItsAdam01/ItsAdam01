# Adam Atienza

I monitor network activity and research the patterns behind anomalous behavior. My work focuses on the entire defense lifecycle: identifying indicators of compromise (IoC), triaging security events, and recommending proactive measures to keep digital infrastructure resilient.

I prioritize visibility and operational logic over abstract theory. Effective defense starts with understanding exactly how a system is supposed to behave: and being the first to notice when it doesn't.
